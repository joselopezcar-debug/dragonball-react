# 🐉 Dragon Ball Explorer

Aplicación web desarrollada con React y TypeScript que consume la API pública de Dragon Ball para explorar personajes del universo de la serie.

## 📋 Descripción

Dragon Ball Explorer es una aplicación que permite visualizar información de personajes obtenida desde una API pública. La aplicación cuenta con navegación entre diferentes rutas y una interfaz construida utilizando componentes de Shadcn UI.

API utilizada:

https://dragonball-api.com/api/characters

## 🚀 Tecnologías utilizadas

- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- Shadcn UI
- Dragon Ball API

## ⚙️ Instalación y ejecución

Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/dragonball-react.git
```

Ingresar al proyecto:

```bash
cd dragonball-react
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en modo desarrollo:

```bash
npm run dev
```

Generar versión de producción:

```bash
npm run build
```

## ✨ Funcionalidades

- Consumo de la API pública de Dragon Ball.
- Página principal con presentación del proyecto.
- Listado dinámico de personajes.
- Ruta independiente para visualizar entidades.
- Navegación mediante React Router.
- Uso de componentes Shadcn UI.
- Diseño responsivo.
